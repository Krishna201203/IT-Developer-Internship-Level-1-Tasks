Projects

Library Management System

This project involved the development of a Library Management System using Python for the frontend interface and SQL for database management. The system was designed to streamline core library operations, enabling users to efficiently manage the library's book collection. Key functionalities implemented include adding new books to the database, issuing books to borrowers, recording book returns, deleting book records, and displaying the current inventory. The Python frontend provides a user-friendly interface for interacting with the system, while the SQL database ensures persistent and organized storage of book information, including titles, authors, ISBNs, and availability status. The outcome of this project is a functional and practical tool that simplifies library management tasks and improves overall efficiency.

Face Recognition System using PCA and MLP

This project focused on developing a robust face recognition system. The system leverages Principal Component Analysis (PCA) for dimensionality reduction to optimize performance and efficiency. A Multi-Layer Perceptron (MLP) classifier was implemented to ensure accurate and precise identification of individuals. The project utilized Python for the frontend development, providing a user-friendly interface for capturing and processing facial images. An SQL database was integrated to securely store and manage facial data, including captured images and corresponding labels. The outcome of this project is a functional face recognition system capable of identifying individuals with a high degree of accuracy, suitable for various applications such as access control or personalized user experiences.
